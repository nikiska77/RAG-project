This guide explains how to add TensorFlow compatibility to an existing Transformers model. You will also learn how to efficiently debug mismatches across MLframeworks. The guide is intended to be used by members of the Hugging Face community. It is not intended as a complete guide, but rather a step-by-step guide to help you with the process of adding Tensor Flow compatibility to your model of choice. For more information on how to use this guide, please visit HuggingFace.com. For help with adding model code to Transformers, visit the Help page.
