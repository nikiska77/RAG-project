FlauBERT is a transformer model pretrained using a masked language modeling objective (like BERT) Like RoBERTa, without the sentence ordering prediction (so just trained on the MLM objective) The model was proposed in a paper by Hang Le et al. The French Language Understanding Evaluation (FLUE) tool can be used to test your knowledge of the French language. For more information on FLUE, visit: http://www.france.org/language-understanding-evaluation/flue-tool.html. The language understanding evaluation tool can also be used by the French National Centre for Scientific Research (NCCR) to test knowledge of French. The code for the FLUE tool is available here: http:www.cnn.com/2013/01/29/science/features/france-language- understanding-exercises-flue.html#storylink=cpy. The tool is also available for English, German, Italian, Spanish, Danish, Norwegian, and other languages. It can be downloaded from the NCCR website here: www.cnsr.org.
