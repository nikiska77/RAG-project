The ByT5 model was presented in a paper by Linting Xue, Aditya Barua, Noah Constant, Rami Al-Rfou, Sharan Narang, MihirKale, Adam Roberts, Colin Raffel. The original code can be found here. The architecture is based on the T5v1.1 model. They only differ in how inputs should be prepared for the model, see the code examples below. The model was contributed by patrickvonplaten. The paper was published in the open-source journal O'Reilly's Machine Learning. It can be downloaded as a zip file from the site here: http://www.oracle.com/machine-learning/Machine-Learning/MachineLearning/ByT5.html. It is available for download in the US and Canada, and in the UK from the U.S. version of the paper is available in the EU version. The U.K. version is available from the company's website: www.orley.co.uk/Machine learning/Machine Learning/By T5. It has been translated into German, French, Spanish, Italian, and other languages.
