The model can be used to predict segmentation masks of any object of interest given an input image. The model is designed and trained to be promptable, so it can transfer zero-shot to new image distributions and tasks.Fine-tuning the model is not supported yet according to the official repository. However, at this time of writing this seems to be not supported. The original code can be found here. It was contributed by ybelkada and ArthurZ. It is free to download and use and is open-source. It can be downloaded and used by anyone. It has been released to foster research into foundation models for computer vision.
