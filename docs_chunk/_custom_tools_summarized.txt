Transformers Agents is an experimental API that is subject to change at any time. Results returned by the agents can vary as the APIs or underlying models are prone to change.Creating and using custom tools and prompts is paramount to empowering the agent and having it perform new tasks. In this guide we'll take a look at: how to customize the prompt, how to use custom tools, and how to create custom tools for the agent. The prompt is structured broadly into four parts: how the agent should behave, explanation of the concept of tools, description of all the tools and a set of examples of tasks and their solution. We'll also look at how the prompt can be best customized.summarize:                  Custom Tools and Prompts for Transformers Agents. The full guide can be found at: Transformers Agents and Custom Tools and Prompts for transformers. The guide can also be downloaded from: http://www.transformers.org/agents/custom-tools-and-prompts-for-transformers-agents.html.
